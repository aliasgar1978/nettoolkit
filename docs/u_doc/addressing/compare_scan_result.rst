
Compare subnet scanner results
============================================

IP scanner output file comparision

.. code-block:: python
	:emphasize-lines: 2

	>>> from nettoolkit.addressing import compare_ping_sweeps
	>>> compare_ping_sweeps(first, second)      # where first and second are two excel file(s)
