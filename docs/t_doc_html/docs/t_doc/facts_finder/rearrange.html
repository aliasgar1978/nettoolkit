5.43.1. table columns

rearrange the columns in appropriate orders

nettoolkit.facts_finder.rearrange.rearrange_tables(clean_file, foreign_keys=None)[source]

rearrange Excel file columns as per interface properties grouping defined globally provide foreign_keys dictionary with list of columns to be extended.

Parameters:

  • clean_file (str) – ouptut clean file name

  • foreign_keys (dict, optional) – custom columns (if any). Defaults to None.
